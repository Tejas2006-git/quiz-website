🎯 Quiz Web Application

📌 Project Description

This is a Quiz Web App built using HTML, CSS, and JavaScript.
Users can select different subjects and attempt quizzes with a timer and score tracking.

---

🚀 Features

- 📚 Multiple Subjects (HTML, CSS, JavaScript, DBMS, etc.)
- ⏱ Timer for each question
- 📊 Score tracking
- 🔄 Restart Quiz option
- 🏠 Go to Home functionality
- 🎯 Randomized answer options (no fixed pattern)

---

🛠 Technologies Used

- HTML
- CSS
- JavaScript

---

📂 Project Structure

QuizApp/
│── index.html
│── style.css
│── script.js
│── README.md

---

▶️ How to Run

1. Download or clone the repository
2. Open "index.html" in your browser
3. Start the quiz
