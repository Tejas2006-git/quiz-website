🎯 Quiz Web Application

📌 Project Overview

The Quiz Web Application is an interactive and user-friendly web app built using HTML, CSS, and JavaScript.
It allows users to select different subjects and test their knowledge through timed quizzes with real-time score tracking.

This project demonstrates strong fundamentals in frontend development, DOM manipulation, and logic building.

---

🚀 Features

- 📚 Multiple Subjects
  Includes HTML, CSS, JavaScript, DBMS, Aptitude, and more.

- ⏱ Timer-Based Questions
  Each question has a countdown timer to improve speed and accuracy.

- 🔀 Randomized Options
  Answer choices are shuffled every time (no predictable pattern).

- 📊 Live Score Tracking
  Score updates instantly after each correct answer.

- 🔄 Restart Quiz
  Users can restart the quiz anytime.

- 🏠 Go to Home
  Easily navigate back to subject selection.

- 📈 Progress Indicator
  Shows current question number (e.g., 3/10).

---

🛠 Technologies Used

- HTML5 → Structure of the application
- CSS3 → Styling, layout, and animations
- JavaScript (ES6) → Logic, DOM manipulation, and interactivity

▶️ How to Run the Project

1. Download or clone this repository
2. Open the project folder
3. Double-click on index.html or open it in a browser
4. Select a subject and start the quiz
